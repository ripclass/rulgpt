# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in tfrules.com, please report it responsibly.

**Email:** security@rulhub.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and aim to provide a resolution timeline within 5 business days.

## Scope

This policy covers:
- tfrules.com (frontend and backend)
- API endpoints at the tfrules.com backend
- Authentication and session management
- Data handling and storage

## Out of Scope

- Third-party services (Supabase, Stripe, Vercel, Render)
- Social engineering attacks
- Denial of service attacks

## Disclosure

We ask that you do not publicly disclose the vulnerability until we have had a reasonable opportunity to address it.
